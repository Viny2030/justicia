Introducción

El Monitor Judicial AR es una plataforma de código abierto para el seguimiento de la eficiencia y la transparencia del Poder Judicial de la Nación Argentina. No tiene publicidad, no recopila datos de usuarios y funciona directamente desde el navegador sin necesidad de instalación ni servidor.

✦
Sin servidor: todos los archivos son HTML + JSON estáticos, alojados en GitHub Pages y ejecutables localmente.
✦
Datos en tiempo real: los JSONs se actualizan automáticamente cada día mediante GitHub Actions que scraplean las fuentes oficiales.
✦
Open source: todo el código fuente está disponible en github.com/Viny2030/justicia bajo licencia MIT.
💡 Este portal no está afiliado al Gobierno Argentino ni al Consejo de la Magistratura. Es un proyecto independiente de transparencia cívica.

Portal de Eficiencia Judicial

Visualiza las estadísticas de causas del Poder Judicial de la Nación por jurisdicción, fuero y tipo de tramitación. Los datos provienen del sistema oficial estadisticas.pjn.gov.ar.

1
KPIs superiores: resumen ejecutivo con totales de causas iniciadas, resueltas, tasa de resolución y demora promedio.
2
Pestañas temáticas: navegá por Trámite, Sentencias, Recursos, Resumen y Comparativa internacional.
3
Tabla de tribunales: cada fila representa un tribunal con sus métricas clave. Hacé clic en un encabezado de columna para ordenar.
4
Exportar CSV: botón en el encabezado para descargar todos los datos en formato CSV compatible con Excel.
📍 La versión actual contiene datos de Comodoro Rivadavia 2024. Próximamente se agregarán las 22 jurisdicciones disponibles del PJN.
👤
Monitor de Magistrados

Seguimiento de los 1.103 magistrados del PJN: designaciones, vacantes, renuncias aceptadas y paridad de género. Datos obtenidos de datos.jus.gob.ar y el Consejo de la Magistratura.

1
Panel KPI: 6 indicadores clave en la parte superior: magistrados activos, vacantes totales, vacantes sin concurso, renuncias aceptadas, paridad de género y magistrados con licencia.
2
Pestaña Magistrados: tabla completa con nombre, cargo, fuero, jurisdicción, tipo (federal/nacional) y estado (titular/subrogante). Filtros por texto en tiempo real.
3
Pestaña Vacantes: listado de cargos sin cubrir, indicando si hay concurso activo o no.
4
Pestaña Renuncias: magistrados que presentaron renuncia aceptada por el Poder Ejecutivo.
5
Pestaña Cruces JGM: cruce de datos con el Monitor de Jefatura de Gabinete para detectar inconsistencias entre fuentes oficiales.

Filtros y búsqueda
1
Búsqueda en tiempo real: escribí en cualquier campo de búsqueda para filtrar la tabla instantáneamente. No hay que presionar Enter.
2
Ordenar columnas: hacé clic en el encabezado de cualquier columna para ordenar ascendente; segundo clic para descendente.
3
Paginación: las tablas muestran 50 filas por defecto. Usá los botones de navegación al pie de cada tabla.
4
Filtros combinados: podés usar varios campos de búsqueda simultáneamente para cruzar jurisdicción + fuero + estado.

Actualización automática

Los datos se actualizan diariamente mediante GitHub Actions. El pipeline ejecuta los scrapers Python y hace commit automático de los JSONs actualizados.

1
scraper_magistrados.py — genera 5 JSONs: magistrados.json, designaciones.json, vacantes.json, renuncias.json y meta_justicia.json
2
scraper_estadisticas.py — actualiza estadisticas_causas.json con los datos más recientes del PJN
3
GitHub Actions ejecuta ambos scrapers a las 03:00 UTC de lunes a viernes y hace git commit + push automático
# .github/workflows/update_data.yml (esquema)
on:
  schedule:
    - cron: '0 3 * * 1-5'   # lunes a viernes 03:00 UTC
jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: pip install -r requirements.txt
      - run: python scraper_magistrados.py
      - run: python scraper_estadisticas.py
      - run: git add *.json && git commit -m "data: update $(date)"

Exportar datos
1
Hacé clic en el botón ⬇ Exportar CSV que aparece en el encabezado de cada portal.
2
Se descarga un archivo .csv con todos los datos de la tabla actual (incluyendo los filtros aplicados).
3
El archivo es compatible con Excel, Google Sheets y cualquier herramienta de análisis de datos.
💡 Los JSONs fuente también están disponibles en el repositorio GitHub y se pueden descargar directamente para uso programático.
